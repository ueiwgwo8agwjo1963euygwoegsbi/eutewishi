Prepare Your Request

To send an SMS, you need to make a POST request with the following information:

Required Fields

  • ‚Ä¢ to: Recipient phone number
  • ‚Ä¢ message: SMS content (max 160 chars)

Optional Fields

  • ‚Ä¢ senderName: Your app name (defaults to "FreeSMS")
2

Phone Number Format

The API accepts Philippine phone numbers in multiple formats and automatically converts them to international format:

‚úÖ Accepted Formats

  • ‚Ä¢ +639123456789 (International)
  • ‚Ä¢ 09123456789 (Local with 0)
  • ‚Ä¢ 9123456789 (Local without 0)
  • ‚Ä¢ 639123456789 (Country code only)

üîÑ Auto-Conversion

All formats are automatically converted to +639123456789 for consistent delivery.

Best Practices

‚úÖ Do's

  • ‚Ä¢ Keep messages under 160 characters
  • ‚Ä¢ Use proper phone number formatting
  • ‚Ä¢ Handle both success and error cases
  • ‚Ä¢ Include timeout handling (10 seconds)
  • ‚Ä¢ Add sender name for better tracking

‚ùå Don'ts

  • ‚Ä¢ Send messages longer than 160 characters
  • ‚Ä¢ Use invalid phone numbers
  • ‚Ä¢ Ignore error responses
  • ‚Ä¢ Exceed 100 requests per 15 minutes
  • ‚Ä¢ Access from unauthorized domains

üåü Real-World Example

Here's a complete example showing how to integrate the SMS API into a web application: 

// Complete SMS sending function with error handling
async function sendSMS(phoneNumber, message, senderName = 'MyApp') {
    try {
        // Validate inputs
        if (!phoneNumber || !message) {
            throw new Error('Phone number and message are required');
        }

        if (message.length > 160) {
            throw new Error('Message must be 160 characters or less');
        }

        // Prepare request
        const response = await fetch('https://jrmph-freesmsapi-bvyo.onrender.com/api/send-sms', {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json'
            },
            body: JSON.stringify({
                to: phoneNumber,
                message: message,
                senderName: senderName
            })
        });

        const result = await response.json();

        if (response.ok && result.success) {
            console.log('SMS sent successfully!');
            return {
                success: true,
                data: result
            };
        } else {
            throw new Error(result.message || 'Failed to send SMS');
        }

    } catch (error) {
        console.error('SMS Error:', error);
        return {
            success: false,
            error: error.message
        };
    }
}

// Usage example
const result = await sendSMS('+639123456789', 'Hello from my app!');
if (result.success) {
    console.log('Message sent:', result.data);
} else {
    console.error('Failed to send:', result.error);
}
