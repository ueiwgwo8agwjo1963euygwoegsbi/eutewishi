Overview & Quick Start

This is a secure, rate-limited public API for sending SMS messages. The API is designed for high reliability and includes robust security features like host-based protection, rate limiting, CORS protection, and input validation.

Base Endpoint

[[BASE_URL]]

Authentication Status

Public Access: No API Key Required (Host-Protected)

Security Features

Endpoint: Send SMS

POST /api/send-sms

Initiates the transmission of an SMS message to a specified mobile number. All inputs are validated for format and length.

Request Body (application/json)

{
    "to": "+639123456789",
    "message": "Hello from JRM FreeSMS API!"
}

Parameters

Parameter Type Req. Description
to string YES Target phone number. Supports international format (+63) or local (09xxxxxxxxx).
message string YES The message content (max 160 characters).

Success Response (200 OK)

{
    "success": true,
    "message": "SMS sent successfully",
    "timestamp": "2025-12-11T20:34:10.000Z",
    "to": "+639123456789",
    "responseTime": "245ms",
    "metadata": {
        "messageLength": 32
    }
}

Error Responses

403 Forbidden (Access Denied)

{
    "error": "Access Denied",
    "message": "This API is only available from authorized domains.",
    "yourHost": "unauthorized-domain.com",
    "allowedHosts": ["jrmph-freesmsapi.onrender.com"]
}

429 Too Many Requests

{
    "error": "Too Many Requests",
    "message": "Rate limit exceeded. Try again later.",
    "retryAfter": 900
}

400 Bad Request

{
    "success": false,
    "error": "Invalid phone number format",
    "message": "Please provide a valid Philippine phone number"
}

Code Examples

Easily integrate the POST /api/send-sms endpoint into your application using common technologies.

cURL

curl -X POST "[[BASE_URL]]/api/send-sms" \
-H "Content-Type: application/json" \
-d '{
    "to": "+639123456789",
    "message": "Hello from JRM FreeSMS API!"
}'

JavaScript (Fetch)

const response = await fetch('[[BASE_URL]]/api/send-sms', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        to: '+639123456789',
        message: 'Hello from JRM FreeSMS API!'
    })
});

const result = await response.json();
console.log(result);

Python (requests)

import requests

url = "[[BASE_URL]]/api/send-sms"
data = {
    "to": "+639123456789",
    "message": "Hello from JRM FreeSMS API!"
}

response = requests.post(url, json=data)
print(response.json())

PHP (cURL)

$url = "[[BASE_URL]]/api/send-sms";
$data = [
    "to" => "+639123456789",
    "message" => "Hello from JRM FreeSMS API!"
];

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
curl_setopt($ch, CURLOPT_HTTPHEADER, ['Content-Type: application/json']);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
curl_close($ch);

echo $response;
